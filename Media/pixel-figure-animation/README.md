# Pixel Figure Animation

This project implements a pixel figure animation with three main actions: running, jumping, and idling. The animations are triggered by user input, providing an interactive experience.

## Project Structure

```
pixel-figure-animation
├── src
│   ├── css
│   │   └── styles.css       # Contains styles for the pixel figure animations
│   ├── js
│   │   └── script.js        # Handles animation logic and user input
│   └── index.html           # Main HTML document for the project
└── README.md                # Project documentation
```

## Features

- **Idle Animation**: The figure remains still until an action is triggered.
- **Running Animation**: The figure runs when the appropriate key is pressed.
- **Jumping Animation**: The figure jumps in response to user input.

## Instructions to Run the Project

1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser.
3. Use the keyboard to control the figure:
   - Press the "Arrow Right" key to run.
   - Press the "Space" key to jump.

Enjoy the pixel figure animation!
